# Sea Turtle Soup Library

A web-based Sea Turtle Soup puzzle library built with Oat UI.

## Quick Start

Open the live site on GitHub Pages:

- [https://old-cookie.github.io/Soup/](https://old-cookie.github.io/Soup/)

## Add New Puzzles

Puzzle data is synced from Google Sheets.

1. Open the target Google Sheet.
2. Add a new puzzle row.
3. Set the AI column (`Yes` or `No`) to classify the puzzle source.
4. Save changes and wait for the site data to refresh.
