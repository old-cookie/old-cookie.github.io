# 海龜湯 JSON 題目產生器

## 目的

依照固定 JSON 結構，創作「全新、未公開、非抄襲」的海龜湯題目，並提供：

- 題目名稱
- 類型（本格或變格）
- 規則（可空）
- 湯面（難以一眼看透）
- 湯底（完整真相與邏輯）

## 何時使用

- 使用者要求「只要 JSON」
- 需要快速生成可遊玩的海龜湯題目
- 需要明確區分本格/變格
- 需要主持人可直接拿來用的湯面與湯底

## 輸出規格

回覆必須是單一 JSON 物件，格式如下：

```json
{
  "題目名稱": {
    "類型": "本格 或 變格",
    "規則": "",
    "湯面": "",
    "湯底": ""
  }
}
```

限制：

1. 不要輸出 JSON 以外的文字。
2. 鍵名必須使用：`類型`、`規則`、`湯面`、`湯底`。
3. `類型` 只能是 `本格` 或 `變格`。
4. `湯面` 要有資訊落差，不能直接暴露真相。
5. `湯底` 需完整解釋事件因果，能對應湯面每個關鍵點。
6. 題目需為原創，不得直接引用常見既有題目。

## 作業流程

1. 先決定題目核心反差（例如：看到的行為與真實動機相反）。
2. 選擇 `本格` 或 `變格`：
   - 本格：現實可發生，因果要合理。
   - 變格：可含超自然或不可能機制，但內部邏輯仍需自洽。
3. 撰寫湯面：保留關鍵資訊、隱藏關聯，讓玩家可透過問答逐步逼近答案。
4. 撰寫湯底：交代人物、時間線、關鍵誤導點與真相。
5. 以 JSON 輸出並自我檢查格式與欄位完整性。

## 品質檢查清單

- 是否為單一合法 JSON 物件
- 是否包含四個固定欄位
- 類型值是否正確
- 湯面是否具謎團感且不劇透
- 湯底是否可完整回收湯面伏筆
- 是否避免與常見經典題高度重複

## 建議互動參數

可接受以下額外條件來客製題目：

- 主題（家庭、校園、醫療、旅遊、科幻）
- 難度（入門、中等、困難）
- 風格（寫實、黑色幽默、溫情反轉）
- 是否加入特殊規則（例如：主持人只能回答「是/否/無關」）
