如何允許混合內容

注意： 允許混合內容可能會降低您的瀏覽安全性。請僅在您信任此應用程式的情況下進行此設定。

Chrome

  1. 點擊網址列右側的鎖頭（或「不安全」）圖示
  2. 選擇「網站設定」
  3. 找到「不安全內容」或「混合內容」選項
  4. 將設定從「封鎖（預設）」改為「允許」
  5. 重新整理頁面

Firefox

  1. 在瀏覽器網址列輸入: about:config
  2. 接受風險警告
  3. 搜尋 security.mixed_content.block_active_content
  4. 將值改為 false (雙擊即可切換)
  5. 重新整理頁面

Edge

  1. 點擊網址列右側的鎖頭（或「不安全」）圖示
  2. 點擊「網站權限」
  3. 找到「不安全內容」並設為「允許」
  4. 重新整理頁面

Safari

  1. 點擊 Safari 選單 > 偏好設定
  2. 選擇「安全性」標籤
  3. 取消勾選「阻止跨站追蹤」選項（注意：這可能影響其他網站的隱私保護）
  4. 重新整理頁面

如果以上方法無法解決問題，請考慮：

  1. 使用其他瀏覽器嘗試
  2. 聯繫系統管理員設置安全的 HTTPS 連接
